File Requirements

This system is designed for use with a preprocessed version of the BraTS2020 dataset. To ensure compatibility, please use the provided dataset linked below.

If you choose to use your own data (not recommended), please ensure it meets the following requirements:

  • File format: .h5 (HDF5)
  • MRI Modalities: T1, T1Gd, T2, T2-FLAIR (4 channels)
  • Image dimensions: 240 × 240 × 4
  • Data type: float64

Note: BraTS datasets are provided in .nii.gz (NIfTI) format and requires conversion to .h5 format and preprocessing to be compatible with this system. If you choose to use your own data, you will need to perform these steps before uploading.

To access the dataset, follow these steps:

  1. Access the dataset using the link provided below
  2. Download and extract the dataset
  3. Navigate to the slice level directory
  4. Select and upload an individual slice file for analysis

Not all slices in the dataset contain tumors and some may be empty. You may need to use the meta-data file (located in the download folder) to identify slices with visible tumors for testing.

Recommended dataset: BraTS2020 Training Data (Kaggle)
