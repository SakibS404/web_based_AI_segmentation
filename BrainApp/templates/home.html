File Requirements

This system is designed for use with the BraTS2020 dataset. To ensure compatibility, please use the provided dataset linked below.

Only upload MRI image files (do not include segmentation mask files).

If you choose to use your own data, please ensure it meets the following requirements:

  • File format: .nii or .nii.gz (NIfTI)
  • Modalities: T1, T1Gd, T2, T2-FLAIR
  • Image dimensions: 240 × 240 × 4
  • Data type: float64

Recommended dataset: BraTS2020 Training Data (Kaggle)
