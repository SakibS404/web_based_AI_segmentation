PLEASE FOLLOW THESE INSTRUCTIONS TO CORRECTLY SETUP THE PROJECT

1: DOWNLOAD AND INSTALL PYTHON 3.10.0   (DO NOT USE A NEWER VERSION DUE TO TENSORFLOW INCOMPATIBILITY)

2: OPEN PROJECT 

3: IN COMMAND PANEL CREATE AND ACTIVATE A VIRTUAL ENVIRONMENT (VENV) USING PYTHON 3.10

4: IN COMMAND PANEL USE:  pip install -r requirements.txt

5: INSTALL API KEY PROVIDED IN  CW SUBMISSION

6: USE COMMAND PANEL TO NAVIGATE TO WebMRI: cd WebMRI

7: RUN THE PROJECT: python manage.py runserver
